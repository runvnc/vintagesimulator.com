Vintage Simulator (VSIM)

Vintage Simulator is a (currently 100% FREE) custom libretro front-end (sharing zero code with RetroArch) with the following features:
  • Implements most of the libretro API
  • Run several emulators at the same time
  • Full Lua scripting API
  • 3D graphics and sound
  • Easily load most 3D models such as 3DS or OBJ
  • Bullet Physics engine
  • Plugins
With VSIM you can create your own virtual collection of vintage computers, consoles, and software, all represented in 3D. Or, using the physics features, you could even build a spacecraft with flight control software written in C64 assembler. You can set up anything you can imagine by importing your own 3D models and writing Lua code.
